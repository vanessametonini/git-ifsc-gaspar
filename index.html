Ola mundo, esta é a oficina de git para o IFSC yay

Download do git

https://git-scm.com/downloads

Para inicializar o git:

git init

Configura meu nome e email no git computador

git config --global user.name "Seu Nomezinho"

Configurar o email no pc

git config --global user.email "vanessametonini@gmail.com"

observação: vc passar --global ou --local

Para ver todas as configurações do git naquele pc

git config --list

Para adicioanr no stage

git add

Registrar as modificacoes

git commit -m "Mensagem"

Para ver os commits da branch

git log

Criar conta no github

Criar um repositorio

Adicionar uma remote origin

git remote add origin https://github.com/vanessametonini/git-ifsc-gaspar.git
